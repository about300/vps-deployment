# VPS 一键部署脚本

这是一个用于快速部署 VPS 服务的自动化脚本，包含以下服务：

- 🔄 **订阅转换** (subconverter) - 支持 VLESS、VMess、Trojan 等协议
- 🛡️ **AdGuard Home** - DNS 广告过滤
- ⚡ **S-UI 面板** - 节点管理
- 🌐 **Nginx** - Web 服务器和反向代理
